##### Exploratory Data Analysis (EDA)



**Purpose**



EDA is the process of examining and understanding a dataset before building a machine learning model. It helps identify patterns, detect anomalies, understand feature relationships, and discover data quality issues.



**Tasks Performed**



• Loaded the dataset using Pandas.

• Examined the dataset structure.

• Displayed summary statistics.

• Checked for missing values.

• Identified duplicate records.

• Analyzed numerical and categorical features.

• Created visualizations such as histograms, box plots, and heatmaps.

• Studied correlations between features.



Libraries Used



• Pandas

• NumPy

• Matplotlib

• Seaborn



Key Learning



• Understood the importance of exploring data before model training.

• Learned how visualizations reveal trends and outliers.

• Gained experience interpreting statistical summaries and feature relationships.
